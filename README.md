# カリキュラム

## レベル1向け
1. コンピュータの基礎
    1. 情報の表現
2. OS
    1. OSの役割
    2. プロセスとスレッド
    3. メモリ管理の基本
    4. ファイルシステムと権限
    5. Linux基本コマンド
3. アルゴリズム
    1. 変数・条件分岐・繰り返し
    2. 計算量（Big-O）の直感
    3. 探索（線形探索・二分探索）
    4. ソート（バブル・選択・挿入）
    5. 疑似コードで手順化
4. ネットワーク
    1. OSI参照モデルとTCP/IPモデルの概要
    2. IPアドレスとサブネットの基本
    3. DNS・DHCP・NTPの役割
    4. HTTP/HTTPSの基本
    5. ping/curl/nslookupの使い方
5. データベース
    1. データベースとは何か（ファイル管理との違い）
    2. 身近なデータベースの例（EC・SNS・業務システム）
    3. テーブル・行・列とは何か
    4. SQL超入門（SELECTとINSERT）
    5. バックアップの必要性と安全な更新
6. ソフトウェア
    1. 開発ライフサイクルの基本
    2. バージョン管理（Git基礎）
    3. テストの種類（単体・結合）
    4. コードレビューの基本観点
    5. リリースとロールバックの基礎
7. セキュリティ
    1. 機密性・完全性・可用性（CIA）
    2. 認証と認可の違い
    3. パスワードと秘密情報の管理
    4. 代表的な脆弱性（SQLインジェクション・XSS）
    5. ログ監視とインシデント初動
