# 概要
このアプリは勉強用アプリ

# 機能
- ユーザー
    - ユーザー作成
    - ユーザー削除
    - ユーザー設定変更

- 認証
    - ログイン(アクセストークン作成)
    - ログアウト(アクセストークン無効化)

- 組織
    - 組織作成
    - 組織削除
    - 組織設定変更
    - グループ作成
    - グループ削除
    - グループ設定変更
    - グループへのユーザー招待
    - グループへのユーザー招待を取り消し
    - アクセスポリシー割り当て

- 教科書
    - 教科書の追加
    - 教科書を閲覧
    - 教科書の一覧表示
    - 教科書の検索
    - 教科書の章を編集
    - タグ追加
    - タグ削除
    - タグ編集機能
