# データモデル

このディレクトリはデータモデル層のディレクトリ。ドメイン層を担う。
以下データモデルを含む

|name           |description                                    |
| :------------ | :-------------------------------------------- |
|Article        |記事を表すクラス                                  |
|Tag            |タグを表すクラス                                  |
|Category       |大項目を表すクラス                                 |
|Subcategory    |中項目を表すクラス                                 |
